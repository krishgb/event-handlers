Events

Mouse-Move & Mouse-Out Events

Whenever you move the mouse around a specific element, the mousemove event will be triggered. When the mouse is out of that specific element, the mouseout event will be triggered.

In this card when you move the mouse, the background color (Hexacolor code eg: #000000 - black color) of the this card will be the height and width of this card. (offsetX & offsetY)

Mouse-Up & Mouse-Down Events

Mouse-Down event occurs when mouse is clicked in a specific element, then the event will be stopped. Mouse-Up event acts as same as mouse-down but it the changes will remain.

Click this card to see the change. Whenever you pressed this card, the background will change to white. After that mouse click, the background will turn to normal and the text color will become white.

Mouse-Over Event

Whenever the mouse is on or over an element, the mouseover event will be triggered..

If your mouse is over the card, the text size of this card will rise to 20 pixels.
